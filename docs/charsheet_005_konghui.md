# 角色美术描述卡 #005 空慧禅师（少林方丈）

## 基础形体
- 年龄外观：六十五岁
- 身高：170cm
- 体型：国字脸，武僧体格，老而弥坚

## 面部特征
- 脸型：国字脸，棱角硬朗
- 眉眼：花白眉毛，眼帘低垂，慈悲相
- 头部：光头，六枚戒疤（顶部）

## 服装描述
- 袈裟：暗赭红 #8B2E2E
- 金边：赤金 #C9A227
- 僧袍内衬：土褐 #5C4A3A

## 武器
- 锡杖：九环，165cm，右手握持

## 配色规范
暗赭红 #8B2E2E / 赤金 #C9A227 / 土褐 #5C4A3A / 肤 #D4B896

## 风格意象
武僧，老而弥坚，慈悲相手段。暗黑武侠风格。

## 与 #001 沈墨渊 对照
| | #001 沈墨渊 | #005 空慧禅师 |
|--|--|--|
| 主色 | 玄黑+深赭 | 暗赭红+土褐 |
| 点缀 | 朱砂 #9B2335 | 赤金 #C9A227 |
| 气质 | 冷峻压迫 | 慈悲威严 |
| 视觉对照 | ✅鲜明对比 |

## 建模重点
- 六枚戒疤位置精确（顶部六点分布）
- 锡杖九环等距分布
- 袈裟金边与主色对比
- 老僧体型（肩宽体沉）
