# 角色美术描述卡 #001 沈墨渊

## 基础形体
- 年龄外观：约三十岁，面容冷峻，棱角分明，无蓄须
- 身高：约 185cm，体态精瘦但不单薄
- 体型：肩宽腰细，四肢修长，肌肉线条内敛
- 站姿：微微仰视，气质压迫感
- 动作特征：动作幅度小、精准，手指无意识敲击椅子扶手

## 面部特征
- 脸型：菱形脸，颧骨微突，下颌线硬朗
- 眉眼：剑眉，眉尾上挑；眼睛细长，瞳色深黑，眼角有细微横纹
- 疤痕：左脸颧骨处有一道斜向浅疤（约 3cm）
- 表情基准：无表情，目光微冷；微笑时只有嘴角动

## 发型与头饰
- 黑发，长及后颈，发顶略高，前额两侧各有一缕散发垂至颧骨
- 束发冠：铜制哑光黑铜，冠形简约，冠顶向后延伸三道火焰线条（明教圣火纹抽象简化）
- 髻簪：左耳后一根铜簪，末端微微上翘

## 服装描述
- 内衬：立领右衽，深灰色细绢，#1A1A1A
- 外袍：宽袖直裾，暗纹锦缎，底色 #2B1B12，火焰暗纹
- 大氅：无袖披风，左肩固定右肩开放，背部朱砂红明教圣火纹徽章

## 武器
- 圣火令（左手握持）：六角形令牌，铜合金，圣火纹 #9B2335 朱砂红

## 配色规范
玄黑 #1A1A1A / 深赭 #2B1B12 / 玄褐 #3D2B1F / 暗铜 #4A3728 / 朱砂 #9B2335 / 赤金 #C9A227 / 肤 #D4B896

## 风格意象
暗黑武侠，明代法门寺铜器 + 《剑雨》反派的冷峻感。非仙侠，不过度飘逸。

## 建模重点
- 疤痕要做细分
- 圣火令火焰纹精度要够
- 背部大氅徽章是远景识别点
- 发冠火焰线条是俯视关键识别点
