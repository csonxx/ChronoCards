## 一、设计概述

### 1.1 核心变化（对比v1.0回合制）

| 要素 | v1.0（回合制） | v2.0（即时ARPG） |
|------|-------------|-----------------|
| 战斗节奏 | 玩家选择指令，等待结算 | 实时操作，无须等待 |
| 普攻 | 单次攻击指令 | 三段连击（轻击1→2→3） |
| 技能 | 选技能列表 | E/Q技能按钮+冷却环 |
| 防御 | 挨打掉血 | 闪避/格挡（无敌帧/完美反击） |
| 元素 | 无 | 六元素附着+反应特效 |
| 资源 | MP魔法值 | 体力/内力/剑意三资源 |

## 二、角色状态区（屏幕左上）

### 2.1 布局结构

```
┌────────────────────────────────────┐
│ [头像] ████████████░░ HP: 850/1000 │ ← 角色头像(40×40)+血条+数值
│ [Lv.12]                            │ ← 等级徽章
├────────────────────────────────────┤
│ [火] [雷] [冰] [毒]                │ ← 元素附着图标（最多4个）
│ [水] [风]                          │ ← 第5-6个换行显示
└────────────────────────────────────┘
```

### 2.2 血条（HP Bar）

| 属性 | 值 |
|------|----|
| 位置 | 头像右侧，水平填充 |
| 尺寸 | 宽 200px，高 16px |
| 背景色 | `#1A1A2E`（深墨色） |
| 填充色 | 渐变 `#4ade80` → `#22c55e`（气血充足） |
| 低血量 | `< 30%` 时变红 `#ef4444`，轻微脉动 |
| 边框 | `1px solid #3d3d5c`（水墨边框） |
| 圆角 | 4px |
| 动画 | 血量变化时平滑过渡（200ms ease-out） |

**HP数值显示：**
```
当前值 / 最大值
例如：850 / 1000
```
- 字体：楷体 bold 12px，白色 `#FFFFFF`
- 位置：血条右侧，紧贴血条
- 低血量时数字闪烁红色

### 2.3 元素附着图标

| 元素 | 图标样式 | 颜色 | 说明 |
|------|---------|------|------|
| 火 | 火焰轮廓 | `#EF4444` 朱红 | 灼烧DOT，持续伤害 |
| 水 | 水波轮廓 | `#3B82F6` 青蓝 | 湿润减速 |
| 雷 | 闪电轮廓 | `#FBBF24` 金黄 | 感电麻痹 |
| 冰 | 冰晶轮廓 | `#A5F3FC` 冰白 | 冻结控制 |
| 风 | 风旋轮廓 | `#A3E635` 翠绿 | 扩散增伤 |
| 毒 | 毒泡轮廓 | `#A855F7` 紫黑 | 持续减防 |

**图标规格：**
- 尺寸：24×24px
- 形状：圆形，带元素轮廓线（2px）
- 背景：`rgba(0,0,0,0.5)` 半透明墨色
- 排列：横向排列，最多显示4个，第5-6个换行
- 新附着：从右侧滑入，带水墨晕染动效
- 消失：元素消退时淡出（opacity 0，300ms）

**元素附着规则：**
- 每次攻击附加元素 → 显示在头像旁
- 同元素再次攻击 → 刷新持续时间
- 元素反应触发 → 图标消失 + 爆发特效
- 最多同时存在 4 种不同元素

## 三、资源条区（屏幕中下部）

### 3.1 三资源条布局

```
┌──────────────────────────────────────────────────────────────┐
│ [体力] ██████████████████████████████░░░░░░░░░░░  100/100   │
│ [内力] ██████████░░░░░░░░░░░░░░░░░░░░░░░░░░░░░   60/100    │
│ [剑意] ████░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░   15/100    │
└──────────────────────────────────────────────────────────────┘
```

### 3.2 体力条（Stamina）

| 属性 | 值 |
|------|----|
| 用途 | 闪避、格挡、冲刺消耗 |
| 最大值 | 100（可随境界提升） |
| 恢复 | 战斗中每秒恢复 5 点 |
| 颜色 | `#F59E0B` 琥珀金 |
| 消耗动画 | 消耗时条减少+右侧飞出"-20"金色数字 |

**格挡消耗：** 每次格挡消耗 15 点  
**闪避消耗：** 每次闪避消耗 10 点  
**体力不足：** 条变红，格挡/闪避按钮变灰禁用

### 3.3 内力条（Qi / Internal Energy）

| 属性 | 值 |
|------|----|
| 用途 | 释放E/Q技能消耗 |
| 最大值 | 100（可随内功境界提升） |
| 恢复 | 受击时回复10点，攻击时回复3点 |
| 颜色 | `#8B5CF6` 紫气（内功颜色） |
| 消耗动画 | 消耗时条减少+紫气飘散粒子 |

**E技能消耗：** 20-40 点（根据技能）  
**Q大招消耗：** 60-80 点（根据技能）  
**内力不足：** 条变暗，技能按钮显示锁定状态

### 3.4 剑意条（Sword Intent）

| 属性 | 值 |
|------|----|
| 用途 | 强化普攻三段连击，累积剑意释放绝技 |
| 最大值 | 100（不可直接恢复） |
| 累积方式 | 普攻命中+5，被击+2，完美格挡+15 |
| 消耗方式 | 剑意满时自动释放绝技，或手动释放Q |
| 颜色 | `#E5E5E5` 银白（剑光色） |
| 特效 | 剑意累积时银色粒子向条汇聚 |

**剑意机制：**
- 0-30：普通普攻（白）
- 31-60：强化普攻，伤害+20%（淡金边框）
- 61-99：精通普攻，伤害+40%，攻速+15%（金边框+剑光）
- 100：绝技自动释放，屏幕银光一闪，巨大范围伤害

### 3.5 资源条通用规格

| 属性 | 值 |
|------|----|
| 高度 | 8px（体力/内力）、6px（剑意） |
| 背景 | `#1A1A2E` 深墨色 |
| 边框 | `1px solid #3d3d5c` |
| 圆角 | 3px |
| 过渡动画 | 200ms ease-out |
| 数值显示 | 右侧，楷体 10px，白色 |

## 四、操作按钮区（屏幕底部）

### 4.1 按钮布局

```
┌──────────────────────────────────────────────────────────────────────┐
│                                                                        │
│   ┌──────┐                                                           │
│   │ 闪避 │    [普攻]    [E技能]    [Q技能]    ┌──────┐              │
│   │ -10  │   ┌────┐     ┌────┐     ┌────┐   │ 格挡 │              │
│   │体力  │   │ 1 │     │ E │     │ Q │   │ -15  │              │
│   └──────┘   │ 2 │     └────┘     └────┘   │体力  │              │
│              │ 3 │                         └──────┘              │
│              └────┘ ← 三段连击指示器                                  │
│                                                                        │
└──────────────────────────────────────────────────────────────────────┘
```

### 4.2 普攻按钮（三段连击指示器）

**外观：**
- 三个小方块 `[1] [2] [3]` 横向排列
- 尺寸：每个 32×32px，方块间距 4px
- 背景：`#1A1A2E`，边框 `1px solid #5a5a8a`
- 数字：楷体 bold 14px，白色

**状态：**
| 状态 | 外观 |
|------|------|
| 可用（待命） | 三个方块均为 `#2a2a4a` 深色 |
| 第一击就绪 | 方块[1] 高亮 `#F59E0B` 金色，带脉动 |
| 第一击中 | 方块[1] 变亮，执行动画 |
| 第二击就绪 | 方块[1] 暗淡，方块[2] 高亮金色 |
| 第三击就绪 | 方块[2] 暗淡，方块[3] 高亮金色 |
| 三击完成 | 三块全暗，500ms后重置回[1]就绪 |
| 连击中断（超时未按） | 300ms内未按则重置，方块全部闪烁一次 |

**三段连击数据：**
| 段数 | 伤害倍率 | 效果 |
|------|---------|------|
| 第一击 | 40% 攻击力 | 快速起手，硬直小 |
| 第二击 | 60% 攻击力 | 中速接上，硬直中 |
| 第三击 | 100% 攻击力 + 击飞 | 爆发收尾，硬直大 |

**普攻附加元素规则：**
- 若角色有内功属性，普攻自动附加元素效果
- 第三击命中时叠一层元素附着

### 4.3 E技能按钮

**外观：**
- 圆形按钮，直径 56px
- 背景：`#1A1A2E`
- 边框：`2px solid #8B5CF6`（紫气内功色）
- 中心显示技能图标（16×16px）
- 技能名"E"标注在图标上方

**冷却环：**
- 覆盖在按钮上方的圆环
- 环宽：4px
- 颜色：`#8B5CF6` 紫色
- 动画：顺时针扫过，从满到空表示冷却进程
- 冷却完毕：环消失 + 按钮短暂发光

**状态：**
| 状态 | 外观 |
|------|------|
| 就绪 | 边框发光 `box-shadow: 0 0 10px #8B5CF6` |
| 冷却中 | 冷却环从右向左扫过，图标半透明 |
| 内力不足 | 边框变灰 `#555`，图标暗淡，显示锁定 |
| 冷却完毕 | 环消失 + 按钮闪烁一次 |

**E技能通用规格：**
- 冷却时间：8-12秒（视技能而定）
- 内力消耗：20-40点
- 可设置3个E技能（玩家切换）

### 4.4 Q技能按钮（大招）

**外观：**
- 圆形按钮，直径 64px（比E技能大）
- 背景：`#1A1A2E`
- 边框：`3px solid #F59E0B`（金色大招色）
- 中心显示大招图标
- 技能名"Q"标注在图标上方

**冷却环：**
- 环宽：5px
- 颜色：`#F59E0B` 金色
- 动画：顺时针扫过

**剑意联动：**
- 剑意达到100时，Q按钮自动发光+脉动
- 屏幕边缘出现"剑意已满！"提示（水墨书法字）
- 按Q触发绝技：银光爆发，全屏AOE

**状态：**
| 状态 | 外观 |
|------|------|
| 就绪（剑意满） | 金色边框发光+脉动，剑气粒子效果 |
| 冷却中 | 冷却环扫过，图标暗淡 |
| 内力不足 | 边框变灰，内力不足提示 |

**Q技能通用规格：**
- 冷却时间：20-30秒
- 内力消耗：60-80点
- 剑意满时自动进入强化状态

### 4.5 闪避按钮

**外观：**
- 圆角方形，40×40px
- 背景：`#1A1A2E`
- 边框：`2px solid #3B82F6`（蓝色闪避色）
- 图标：闪电/燕子符号（轻功意象）

**状态：**
| 状态 | 外观 |
|------|------|
| 就绪 | 蓝色边框发光 |
| 体力不足（<10） | 边框变灰，按钮禁用，图标暗淡 |
| 闪避中 | 按钮缩小（scale 0.9）+蓝光拖尾 |

**闪避反馈：**
- 闪避成功触发无敌帧：角色半透明闪烁
- 屏幕轻微震动（2px，100ms）

### 4.6 格挡按钮

**外观：**
- 圆角方形，40×40px
- 背景：`#1A1A2E`
- 边框：`2px solid #22C55E`（绿色防御色）
- 图标：盾牌/剑交叉符号

**状态：**
| 状态 | 外观 |
|------|------|
| 就绪 | 绿色边框发光 |
| 体力不足（<15） | 边框变灰，按钮禁用 |
| 格挡中（持续按） | 边框高亮，角色身前出现水墨护盾 |
| 完美格挡 | 参见第五节特殊反馈 |

**格挡机制：**
- 持续按压格挡键：角色进入格挡姿态
- 松开格挡键：解除格挡
- 格挡成功：减少 80% 伤害
- 完美格挡：在敌人攻击命中瞬间格挡，触发反击

## 五、特殊反馈动效

### 5.1 完美格挡（Perfect Block）

**触发条件：** 敌人攻击命中前 200ms 内按下格挡

**视觉效果：**
```
┌─────────────────────────────────────────────────────┐
│                                                     │
│            【 水墨冲击波 】                          │
│                                                     │
│     ╱╲                                            │
│    ╱  ╲        "完美格挡！"                        │
│   ╱────╲       水墨大字，白色描边                   │
│   │    │       屏幕轻微震动（3px，150ms）           │
│   ╲    ╱       格挡瞬间金光闪烁                     │
│    ╲  ╱        敌人攻击被反弹（红色伤害数字）        │
│     ╲╱                                            │
│                                                     │
└─────────────────────────────────────────────────────┘
```

**动画序列：**
1. 格挡瞬间：格挡按钮放大 `scale(1.3)` + 金光
2. 角色身前：浮现水墨护盾（半透明墨色圆盾）
3. 敌人攻击命中：金光爆发 + 屏幕震动
4. 反击伤害：敌人头顶飘出白色"反击！+200"伤害数字
5. 反馈消失：水墨护盾淡出（300ms）

**音效提示（文字标注）：** "叮！——"

### 5.2 完美反击（Perfect Counter）

**触发条件：** 完美格挡后 500ms 内按普攻

**视觉效果：**
```
┌─────────────────────────────────────────────────────┐
│                                                     │
│           【 剑光一闪 】                             │
│                                                     │
│    ～～～～～～～～～～～～～～～                   │
│    ～　  "反击绝杀！"　  ～   ← 水墨书法白字        │
│    ～～～～～～～～～～～～～～                    │
│    ～～～～～～～～～～～～～～                    │
│                                                     │
│    银光从角色剑尖划向敌人，造成 300% 伤害            │
│    触发剑意+15                                       │
│                                                     │
└─────────────────────────────────────────────────────┘
```

**动画序列：**
1. 银光线条：从角色位置划向敌人（100ms）
2. 剑光爆发：命中处银光爆开（150ms）
3. 伤害数字：巨大白色"反击！300"（200ms渐显）
4. 敌人硬直：敌人短暂僵住（300ms）
5. 剑意获得：右侧飘"+15剑意"银色字样

### 5.3 元素反应特效（水墨风）

**六种元素反应设计：**

| 反应名 | 触发 | 视觉特效（水墨风） |
|--------|------|------------------|
| **蒸发** | 火 + 水 | 墨点沸腾蒸发，白烟升腾，爆发圆晕 |
| **融化** | 冰 + 火 | 冰晶碎裂，墨色冰块融化为水墨滴落 |
| **超导** | 雷 + 冰 | 银白电弧在冰面上扩散，线条如电书法 |
| **感电** | 雷 + 水 | 金色电弧从水波中跳跃，书写式电光 |
| **扩散** | 风 + 火/雷/毒 | 墨点被风卷起，向外扩散成圆形水墨晕 |
| **冻结** | 水 + 冰 | 水墨凝固成冰雕效果，蓝白渐变，冰裂纹 |

**特效动画示例 - 蒸发（水墨风）：**
```
状态1（附着火）：墨点呈红色，缓慢燃烧
状态2（附着水）：墨点呈蓝色，水波荡漾
状态3（触发蒸发）：
  - 水墨沸腾动画（100ms）
  - 白烟升腾粒子（200ms）
  - 爆发圆晕向四周扩散（300ms）
  - 伤害数字："蒸发！+150"
  - 水墨圆晕淡出（200ms）
```

**元素反应优先显示：**
- 屏幕中央偏下显示反应名称（大字水墨书法）
- 反应特效叠加在战斗场景中
- 多个反应同时触发时，垂直堆叠显示

### 5.4 普攻三段连击视觉反馈

**第一击：**
- 角色出剑，水墨刀光从右向左划过
- 伤害数字："60"（白色）
- 敌人轻微后退

**第二击：**
- 连续第二道水墨刀光
- 伤害数字："90"（淡金色）
- 敌人中度后退

**第三击：**

## 八、与其他界面的接口

### 8.1 切换说明

- **进入战斗：** 从世界探索（S3）触发遭遇 → 屏幕水墨淡入 → 进入S5战斗
- **脱离战斗：** 战斗胜利 → 水墨淡出 → 返回S3世界探索
- **战斗失败：** 屏幕变红 → 重伤结算 → 可选重新挑战或返回

### 8.2 数据传递

| 数据项 | 来源 | 用途 |
|--------|------|------|
| 角色HP/等级 | S3角色状态 | 显示在战斗UI |
| 元素属性 | 角色内功配置 | 决定普攻附加元素 |
| 技能列表 | 角色已解锁技能 | 填充E/Q按钮 |
| 敌人数据 | 遭遇生成 | 展示敌人HP条（头顶） |
