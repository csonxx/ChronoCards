**优先级：🔴 P0（首个启动）**

### 1.2 [ ] 场景节点（Location Marker）
- Location 节点图标：区分 city/town/village/wilderness/dungeon/inn/special
- 已解锁 vs 未解锁视觉状态

- 点击节点发起 navigate（调用 POST /players/{id}/location/navigate）

### 1.5 [ ] 当前场景信息面板
- 场景名称、氛围描述、大区

### 1.6 [ ] 世界概览 HUD
- 当前大区名称
- 激活事件提示
- 已解锁地点计数 / 总数

### 2.1 [ ] 茶馆NPC发牌员

**角色定位**：江湖消息汇聚之地，主线剧情触发入口

#### 外观图标
| 元素 | 规格 |
|------|------|
| 主图标 | 掀帘茶馆门面（水墨画风格），屋檐下挂"茶"字幌子 |
| 辅助图标 | 茶杯热气袅袅（动态，CSS动画） |
| 悬挂标识 | 竹制令牌，上刻"说书人"三字（位于茶馆门口） |
| 风格配色 | 土黄 #C4A35A + 暗青 #4A5568 + 茶绿 #6B8E6B |
| 尺寸规范 | 主图标 96×96px，悬停放大至 112×112px |

#### 发牌员立绘（说书人NPC）
- **外观**：老者/中年江湖客，书生打扮但略带风尘气息
- **服装色**：灰青长衫 #5A6B7A + 补丁口袋（暗示情报来源）
- **配饰**：腰间一枚铜铃（摇动表示有话要说），手执折扇
- **情绪**：常态平和，激活时眼神锐利，冷却时闭目养神

#### 状态定义
| 状态 | 视觉表现 |
|------|----------|
| **空闲（Idle）** | NPC背对或侧身，茶杯热气袅袅，铜铃静止，悬停显示"暂无新消息" |
| **激活（Active）** | NPC正脸出现，折扇展开，眼中有光，铜铃微晃，头顶金色感叹号气泡 |
| **冷却中（Cooldown）** | NPC背影+低头，茶杯无热气，显示倒计时砂漏（⏳N分钟），不可交互 |

#### 交互方式
- **触发方式**：点击茶馆节点 → 进入茶馆场景 → NPC自动激活/弹出对话气泡
- **对话气泡**：Char006同款气泡，土黄底 + 暗青边框，左下角缺角（丐帮风格）
- **气泡内容**：「江湖风雨，今日有何贵干？」/「有一桩要事相告……」
- **发牌流程**：对话完成 → 自动触发抽卡 → 卡牌类型 = 主线推进卡 / 角色主导卡 / 世界碎片卡

#### 卡牌类型关联
| 触发场景 | 卡牌类型 |
|----------|----------|
| 主线剧情推进 | 主线推进卡（60%）+ 世界碎片卡（30%）+ 空白卡（10%）|
| NPC对话交付 | 角色主导卡（70%）+ 世界碎片卡（20%）+ 遭遇NPC卡（10%）|
| 情报打听 | 角色主导卡（40%）+ 遭遇NPC卡（40%）+ 成长学习卡（20%）|

### 2.2 [ ] 悬赏板发牌员

**角色定位**：江湖恩怨告示处，支线任务与奖励获取入口

#### 外观图标
| 元素 | 规格 |
|------|------|
| 主图标 | 斑驳木牌（卷轴展开状），边缘有竹钉 |
| 表面样式 | 羊皮纸/宣纸质感，仿旧黄褐 #D4B896 |
| 标题标识 | 顶部横书"悬赏"二字（碑刻体/楷书），朱砂红 #C94A4A |
| 辅助图标 | 告示下方垂落红绳，绳端系小铜铃 |
| 风格配色 | 羊皮黄 #D4B896 + 朱砂红 #C94A4A + 深木棕 #6B4423 |
| 尺寸规范 | 主图标 96×96px，激活时展开为 128×160px 悬赏画卷 |

#### 悬赏板立绘
- **外观**：多张悬赏令叠贴的木板（近景可见3-4张告示边缘）
- **纸张效果**：泛黄卷边，有撕裂痕迹和墨渍
- **激活提示**：当前可接任务对应的悬赏令微微发光（金边），其余暗淡
- **冷却效果**：铜铃静止，板面整体暗化，显示"下次刷新：⏳N分钟"

#### 状态定义
| 状态 | 视觉表现 |
|------|----------|
| **空闲（Idle）** | 悬赏板静态，无铃响，有可接任务时微微发光（暗示可交互）|
| **激活（Active）** | 当前悬赏令展开高亮，红绳铜铃轻晃，顶部出现"新！"角标 |
| **冷却中（Cooldown）** | 板面灰度化，铜铃静止，显示刷新倒计时，不可点击 |

#### 交互方式
- **触发方式**：点击悬赏板节点 → 进入悬赏板场景 → 展示当前任务列表
- **任务卡展示**：点击单张悬赏令 → 展开详情面板（任务名/难度星级/奖励/时限）
- **任务难度星级**：★~★★★★★，一星绿/二星蓝/三星黄/四星橙/五星红
- **接取/放弃**：接取按钮（朱砂红底）+ 放弃按钮（暗灰底），二次确认弹窗
- **发牌流程**：接取任务 → 完成交付 → 触发支线探索卡 + 宝藏卡（奖励）

#### 卡牌类型关联
| 任务类型 | 卡牌类型 |
|----------|----------|
| 支线跑腿 | 支线探索卡（50%）+ 空白卡（30%）+ 遭遇NPC卡（20%）|
| 战斗挑战 | 遭遇战斗卡（60%）+ 支线探索卡（30%）+ 空白卡（10%）|
| 情报收集 | 角色主导卡（40%）+ 遭遇NPC卡（40%）+ 世界碎片卡（20%）|
| 奖励交付 | 宝藏卡（100%，必定触发奖励发放）|

| 风格配色 | 暗铁灰 #4A4A5A + 血红 #8B2020 + 黑青 #1A202C |
| 尺寸规范 | 主图标 96×96px，激活时刀身有红色血光脉动效果 |

#### 敌人立绘（通用模板）
- **外观**：根据敌人类型变化（流寇/门派弟子/野兽/BOSS）
- **危险等级HUD**：头顶血红色标签，显示 ★~★★★★★
- **战斗风格标识**：刀/剑/拳/暗器，对应图标（64×64px）
- **通用色调**：血红 #8B2020 强调危险，深灰 #4A4A5A 作为主调

#### 状态定义
| 状态 | 视觉表现 |
|------|----------|
| **空闲（Idle）** | 敌人背影/侧影，双刀静止，血迹暗化，不可交互 |
| **激活（Active）** | 敌人正身出现，血红气息环绕，刀身血光脉动，脚下暗红光环 |
| **冷却中（Cooldown）** | 敌人消散为黑烟，只留双刀插地，显示⏳倒计时 |

#### 交互方式
- **触发方式**：危险区域移动时随机遭遇 → 自动弹出敌人遭遇UI
- **遭遇选项面板**：三按钮横排
  - ⚔️ **战斗**（血红按钮）：直接触发S5_ARPG_v2战斗界面
  - 🧠 **智取**（蓝灰按钮）：触发审问/策略选项，消耗情报值
  - 🏃 **逃跑**（暗青按钮）：成功率判定，失败则强制进入战斗
- **审问交付**：智取成功后 → 触发角色主导卡（审问结果）+ 可能获得情报道具
- **战斗交付**：战斗胜利后 → 触发遭遇战斗卡 + 宝藏卡（战利品）

#### 卡牌类型关联
| 遭遇结果 | 卡牌类型 |
|----------|----------|
| 战斗胜利 | 遭遇战斗卡（70%）+ 宝藏卡（30%，战利品）|
| 智取成功 | 角色主导卡（60%）+ 遭遇NPC卡（30%）+ 成长学习卡（10%）|
| 逃跑成功 | 空白卡（80%）+ 支线探索卡（20%，绕路遭遇）|
| 战斗失败 | 孤寂卡（100%，重伤/反思）|

### 发牌员通用设计规范

#### 全局色彩规范（丐帮江湖风）
```
主色（土黄）：#C4A35A  — 用于中立/和平场景（茶馆/客栈/商贩）
辅色（暗青）：#4A5568  — 用于紧张/危险场景（敌人）
点缀（朱砂红）：#C94A4A  — 用于奖励/激活/紧急
资源（金）：#D4A843  — 用于货币/高价值物品
草药绿：#6B8E6B  — 用于恢复/休息相关
茶绿：#5D8A66  — 用于丐帮专属标识
```

#### 全局状态系统（5种发牌员通用）
| 状态 | 颜色标识 | 动效 |
|------|----------|------|
| 空闲（Idle） | 灰度化 30% | 微弱呼吸动画 |
| 激活（Active） | 全彩 + 高亮描边 | 脉冲光晕 / 摇曳动画 |
| 冷却中（Cooldown） | 灰度化 70% | ⏳倒计时 + 静止无动画 |

#### 对话气泡规范（5种发牌员通用）
- **气泡底色**：`#E8D5A3`（土黄宣纸色）
- **气泡边框**：`#3D4A5C`（暗青灰）
- **气泡角**：左下角缺角（8px，直角缺，丐帮风格标识）
- **气泡阴影**：`0 4px 12px rgba(0,0,0,0.3)`
- **气泡内边距**：16px
- **气泡最大宽度**：320px
- **气泡文字**：深棕 `#3D2B1F`，行高1.5
- **豪爽/激活状态**：边框变为 `#D4A843`（暖金色）
- **冷却/警告状态**：气泡边框变为 `#8B2020`（暗红色）

### 7.2 [ ] 抽卡动画规格

#### 7.2.1 牌堆3D展示
- **视觉**：半透明3D牌堆（20~30张牌厚度），透视效果，牌背朝上
- **透视角度**：俯视约15°，略微倾斜，牌堆顶部卡片可见
- **牌背样式**：水墨卷云纹，中心印章"江湖"二字，四角竹节纹
- **牌背配色**：深靛蓝 #1A2A4A + 金色烫印 #D4A843
- **动画**：牌堆有轻微浮动动画（Y轴上下3px，周期2s）

#### 7.2.2 飞牌轨迹动画
- **触发时机**：玩家点击"抽卡"按钮 或 发牌员对话完成自动触发
- **轨迹**：卡牌从牌堆顶部斜向前飞出（Z轴推进 + Y轴上升 + X轴微偏）
- **时长**：600ms，缓出曲线（ease-out-cubic）
- **飞牌姿态**：飞行时保持背面朝上，有轻微旋转（±5°）
- **飞牌残影**：拖尾效果（3张半透明残影，渐隐）

#### 7.2.3 翻转动画（Card Flip）
- **翻转轴**：横向翻转（rotateY）
- **时长**：500ms，缓入缓出（ease-in-out）
- **翻转节奏**：
  - 0~250ms：背面 → 侧边（90°），卡牌缩小至80%高度（3D压缩）
  - 250ms：切换展示面（背面→正面），卡组重置
  - 250~500ms：侧边 → 正面（90°→180°），恢复100%高度
- **正背切换瞬间**：有轻微光效闪烁（金色光芒从中心向外扩散，100ms）
- **翻转时背景**：翻转区域有聚光灯效果（ radial-gradient 聚焦在卡牌上）

#### 7.2.4 正面展示动画
- **弹入效果**：翻转完成后卡牌轻微弹跳（scale 1→1.05→1，200ms）
- **光晕脉动**：卡牌边框有类型色光晕脉动（周期1.5s，无限循环）
- **等待交互**：正面展示停留 3s（可配置），之后"选项按钮"淡入出现

#### 7.2.5 多抽连动（批量抽卡）
- **连抽动画**：每隔 800ms 抽出一张，连抽时前一张向左滑出消失（200ms），新卡从牌堆飞出
- **多张排布**：最多同时屏幕显示 1 张正在翻转 + 左侧列队（最多3张已抽卡牌缩略图）
- **汇总展示**：批量抽完后，弹出"本轮抽卡汇总"面板（横向排列所有抽到的卡）
